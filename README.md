# GameRecommender 🎮
Дипломный проект по специальности 09.02.09 «Веб-разработка» (АлтГТУ).
Веб-сервис персонализированного подбора видеоигр.
## Использованный стек:
* **Frontend:** HTML5, CSS3, JavaScript (DOM, динамическое переключение цветов
* **Backend:** Python (Flask), размещение на хостинге PythonAnywhere
